HOOKS EN REACT

Los Hooks en React permiten usar funcionalidades como el estado y los efectos secundarios en componentes funcionales. El hook useState permite agregar y manejar el estado dentro de un componente funcional, proporcionando una forma sencilla de actualizar y acceder a valores. Por otro lado, useEffect se utiliza para realizar efectos secundarios, como la ejecución de código después de que el componente se renderiza o cuando ciertas dependencias cambian, lo que es útil para tareas como la obtención de datos o la manipulación del DOM. Estos dos hooks simplifican la gestión del estado y los efectos, eliminando la necesidad de componentes de clase y haciendo los componentes funcionales más poderosos.
