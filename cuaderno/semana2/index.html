HTML Y CSS

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear páginas web. Funciona estructurando el contenido de la web mediante etiquetas, las cuales organizan elementos como encabezados, párrafos, imágenes, enlaces y más. Básicamente, es el esqueleto de una página web. Por otro lado, CSS (Cascading Style Sheets) es el lenguaje que se encarga de darle estilo y formato a esa estructura HTML. Mediante CSS, se puede modificar la apariencia visual de los elementos, controlando aspectos como colores, tamaños, márgenes, fuentes y el diseño en general. Además, CSS permite hacer diseños adaptativos o "responsive", ajustando la presentación del contenido en diferentes dispositivos, como móviles o tablets.
