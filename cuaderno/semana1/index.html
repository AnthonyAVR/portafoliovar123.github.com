FUNDAMENTOS DE LA TECNOLOGÍA WEB

Los fundamentos de las tecnologías web son el conocimiento clave para entender cómo funcionan las aplicaciones en internet. Esto incluye el uso de protocolos como TCP/IP y HTTP, que permiten el envío y la recepción de información, así como el sistema DNS, que convierte los nombres de dominio en direcciones IP. Las tecnologías web básicas abarcan HTML para dar estructura a las páginas, CSS para su estilo, y JavaScript para hacerlas interactivas. También es importante conocer los roles en el desarrollo web: frontend (encargado de la parte visual), backend (centrado en la lógica del servidor) y fullstack (que combina ambos).
